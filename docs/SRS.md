# Software Requirements Specification
## CampusRide: Student Transport Booking System v1.0

### 1. Introduction
This SRS defines requirements for CampusRide, a mobile system for FBC students to book transport.

### 2. Functional Requirements
FR-01: System shall allow students to create ride requests
FR-02: System shall notify drivers of new requests
FR-03: System shall calculate estimated fare based on distance

### 3. Non-Functional Requirements
NFR-01: Response time < 3 seconds
NFR-02: System available 95% during school hours
