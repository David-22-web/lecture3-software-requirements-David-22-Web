# Traceability Matrix - CampusRide

This matrix links User Stories to SRS Requirements and Test Cases.

| User Story ID | Description | SRS Requirement ID | Test Case ID | MoSCoW Priority |
| --- | --- | --- | --- | --- |
| **US-01** | As a student, I want to book a ride with pickup/dropoff location so that I get to class on time | FR-01 | TC-01 | Must Have |
| **US-02** | As a driver, I want to accept or reject a ride request so that I control my schedule | FR-02 | TC-02 | Must Have |
| **US-03** | As a student, I want to see estimated fare before booking so that I can budget | FR-03 | TC-03 | Should Have |
| **US-04** | As admin, I want to generate daily ride reports so that I can pay drivers | FR-04 | TC-04 | Should Have |
| **US-05** | As a student, I want to rate the driver after ride so that service quality improves | FR-05 | TC-05 | Could Have |
